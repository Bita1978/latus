
DirHash by Mounir IDRASSI (mounir@idrix.fr) Copyright 2010
Recursively compute hash of a given directory content in lexicographical order.
It can also compute the hash of a single file.

Supported Algorithms : SHA1, SHA256, SHA384, SHA512
Using OpenSSL

Using SHA512 to compute hash of "hash" ...
test\data\hash\a.txt
test\data\hash\b.txt
