Latus (lat.us) - Making Cloud Storage Services Secure for Personal Files

Motivation

Many people use cloud storage services such as Dropbox, iCloud, Microsoft's One Drive, etc. for their personal file syncing needs.  While these services are quite convenient, there is still the risk of having your files 'exposed' if there would happen to be a security flaw or hack.  Many personal files are 'sensitive' in nature, such as financial records.  Even though these services claim to be secure, there have certainly been many data 'breaches', especially recently.  Even large corporations have fallen prey to these attacks and it seems that no entity is immune.  Also, 'phishing' attacks can expose data in an otherwise secure system.  An additional level of security is needed to ensure it is safe to use these cloud storage and syncing services for your sensitive files.

Technique

Latus encrypts your files before sending them to the syncing folders that are used by your cloud storage provider.  A parallel latus folder is created where your files are stored in their normal (unencrypted) state.  The latus application monitors the latus folder.  When a new or modified file is detected it is encrypted and copied to a special .latus cache folder (that users generally do not use) in the cloud storage folder.  The diagram below shows this:



Implementation

Latus is an open source Python application that encrypts your files before they are sent to your cloud storage provider.  It is available at https://github.com/latusrepo/latus .

Competitors and Related Work

Keepass is an open source password management application.  It encrypts passwords before storing them in a local database file.  It only does password (and associated comments), however.
